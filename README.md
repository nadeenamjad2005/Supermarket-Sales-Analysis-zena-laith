# Supermarket-Sales-Analysis-zena-laith
Supermarket Sales Analysis

Project Description

This project focuses on analyzing supermarket sales data using Python to better understand customer behavior, compare sales across branches, and study the impact of customer type and payment methods on sales performance.

Tools & Libraries
	•	Pandas – Data loading, cleaning, and manipulation
	•	NumPy – Numerical computations
	•	Matplotlib – Data visualization
	•	Seaborn – Advanced and aesthetic visualizations

⸻

Project Steps
	1.	Load the dataset and inspect its structure.
	2.	Clean the data and verify that there are no missing values.
	3.	Convert date and time columns into appropriate formats for analysis.
	4.	Perform Exploratory Data Analysis (EDA):
	•	Analyze sales by branch.
	•	Compare customer types and genders.
	•	Study payment methods.
	5.	Create visualizations to identify trends and relationships.
	6.	Summarize insights and key findings from the analysis.
